# GOTRAYMON

![GitHub commit activity](https://img.shields.io/github/commit-activity/w/core6quad/gotraymon)


a simple cross-platform system monitor that lives in your tray, written in go


## Build

On windows: use the batch file, it automatically builds with a flag
to disable the console window.

On Linux OR MacOS: `go build .`
i don't think that linux will work.
